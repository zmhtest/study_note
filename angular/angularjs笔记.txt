IT营angular js教程笔记

谷歌的一款开源框架

学习版本angular4.x

angular2.0以后都是基于typescript，他是微软开发的编程语言。

作用：
1.APP和微信单页面应用。
2.借助ionic、react native开发跨平台的原生APP
3.开发桌面应用

angular、react、Vue之间的区别：谷歌、Facebook、个人主导的项目。

Vue只关注页面，angular和react都关注安排和web。

安装angular：
1.安装node js
2.安装cnpm，npm install -g cnpm --registry=https://registry.npm.taobao.org

3.安装脚手架：cnpm install -g @angular/cli

4.利用脚手架创建项目：ng new my-app
	会创建项目和安装所有依赖！！！

5.进入my-app目录，执行cnpm install 安装所需要的依赖，此时生成一个node_modules文件夹

6.启动服务：ng serve --open

创建组件：
	ng g component components/header
