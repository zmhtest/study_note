	def fun(param1，param2):

python中的每个函数都有返回值，如果没有使用return语句，则返回值为 None 

python中不需要给任何东西（返回值，参数）显式的指定类型，这一点与java不一样

注意：默认值一定是放在参数列表的最后！！！

### 代码缩进

python中唯一的定界符就是一个冒号（:）和代码自身的缩进。

缩进表示一个代码块的开始，非缩进表示一个代码的结束。

Python使用回车符来分割语句，使用一个冒号和缩进来分割代码块。

### 异常处理

Python使用try...except块来处理异常，使用raise语句来抛出异常。

Java和C++使用try...catch块来处理异常，使用throw语句来抛
出异常。

if size < 0:
	raise ValueError('number must be non‐negative')

异常是用类来实现的，这个raise语句事实上正在创建一个ValueError类的实例并传递'number must be non‐negative'到它的初始化方法里面。

模块是对象，并且所有模块都有一个内置的属__name__。
