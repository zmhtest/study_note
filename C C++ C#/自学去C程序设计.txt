> # 相当于define，用来引入一个头部文件 #include <stdio.h>

<stdio.h>是编译器安装目录下的一个文件，里面包含各种预定义函数，比如printf() scanf()...只有引入相应的头文件才能使用相关的函数

int main(){return 0;}这是配套使用的，return 0 表示程序运行结束

c语言是强类型的，变量在定义是必须声明类型 int i;

scanf("%d",&k);表示接受用户传入的整型数据，并把它存放在变量k所指向的内存空间，所以k值随之改变，并不是原来的k值。

%d表示整型，%f表示浮点型，这是格式化输出,这里的%d %s %f只是占位符，会被后面的替换掉

double比float更精确，在C语言中没有String关键字，要表示字符串只能用char[]数组来表示char a[]="dssafg"

常量的定义：
const 
#define 常量名 

常量一旦定义就不能再改变，只能是最初所赋的那个值

#define zmh "朱明浩" 这是用来定义字符串常量的
const int x = 12;

头部不能有分号，只有在函数体部分才能有分号

格式化输出中：前面的占位符和后面的值是一一对应的！除了占位符和\n等转义字符其他的都是原样输出的

数组名称就是代表数组的地址，无需再用&符号，如果是普通变量就需要用地址符

在codeblock编辑器中修改了字符编码，还要在编辑界面重新输入，并保存才能生效

scanf("%s%d",str,&i);需要接收多少个数就写多少个格式占位符，后面对应多少个变量名(表示变量的地址) 往变量中放值要用地址符，取出或者打印出变量值无需地址符
