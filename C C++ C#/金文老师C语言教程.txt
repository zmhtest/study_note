# 金文老师C语言教程

计算机语言：
	低级语言：接近硬件
	高级语言：远离硬件

机器语言：计算机能够识别的二进制指令

对C语言来说：
	源程序：用高级语言写的程序
	目标程序：计算机能识别的二进制代码
	编译程序：将源程序编译成目标程序的一个中间程序

流程图：  

	传统的结构化流程图：
	开始框（椭圆）
	处理框（矩形）
	输入输出框（平行四边形）
	判断框（菱形）
	流程线（箭头）
