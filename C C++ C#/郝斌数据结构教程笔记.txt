# 郝斌数据结构学习笔记

高一凡：很厉害的一个人，将数据结构的所有算法都用c语言来实现了！！！

数据结构的定义：把大量复杂的数据类型（个体）和存储结构（个体之间的关系）保存到内存中，以及在此基础上实现的功能（比如查找某个元素），这个功能就是算法

衡量算法的标准：
	1. 时间复杂度
		大概程序要执行的次数，而不是时间，因为硬件有差异
	2. 空间复杂度
		算法执行过程中大概占用的最大内存
	3. 难易程度
		要尽量让更多的人看得懂，可读性高
	4. 健壮性
		很少出错

数据结构的特点：

数据结构是我们软件中最核心的课程，在很多语言中都可以找到数据结构的影子

所谓的栈内存和堆内存，实际上都是分配内存的方式不一样

函数的调用：压栈和出栈

牵涉到时间的问题：队列

编译原理：树

数据库：数据库就是我们数据结构的简化版 

	字段vs变量 
	记录vs个体 
	表vs相同个体的集合 
	外键vs个体之间的关系

什么是程序：数据的存储+数据的操作+可以被计算机执行的语言

数据结构是一门会慢慢发酵的课程！！！
