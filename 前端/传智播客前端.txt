# HTML部分

现在的网页不仅仅是一个页面，而是相当于一个应用程序！！！

UI设计师+前端开发工程师

页面-->界面

<!DOCTYPE>声明这是一个HTML文档
<html>
	<head>
		<title>标题</title>
		<meta charset='utf-8'>
	</head>
	<body>

	</body>
</html>

一个HTML文件就是一个网页

HTML的文档类型：HTML1.0和HTML5
新建HTML1.0的快捷方式：html:xt+tab
新建HTML5的快捷方式：html:5+tab或者！+tab

HTML文档规范：可遵守可不遵守
1.所有的标签必须全部小写
2.所有的属性值用引号括起来
3.所有标签必须闭合 <br /> 空格 /闭合
4.img标签必须加alt属性
<img src="" alt="风景图">
图片找不到时，显示的文字
有利于爬虫收集图片
盲人的读图软件

注释： c+tab
<!-- 只是对代码的标注 -->

标题 ：h1到h6
注意：搜索引擎会使用标题将网页的结构和内容编制索引，提高网页的辨识度

空格：&nbsp;
如果是多个空格可以用样式来控制
用键盘敲入的空格键，无论多少个都是算一个空格的！

多使用语义化的标签，有利于搜索引擎收录，提升排名

相对路径 ./images  
其中 ./ 是可以省略的
../表示上一级目录

超链接中的 title属性，可以在鼠标放在链接上时显示title的值
<a href="http://itcast.com" title="go to itcast>传智播客</a>

给图片添加超链接：<a href="http://www.baidu.com" title="百度logo"><img src="baidulogo.png" alt="百度图片"></a>

超链接的缺省值： href="#"
实际作用：连接到当前页面顶部

<a href="JavaScript:;">缺省值，啥都不做</a>
超链接里面可以写js语句，然后以分号结束

定义页面内滚动跳转
在超链接的地址里面写 #id
<a href="#id1"></a>
<a href="#id2"></a>
<a href="#id3"></a>
<a href="#id4"></a>

<p id="id1"></p>
<p id="id2"></p>
<p id="id3"></p>
